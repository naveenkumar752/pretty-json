# 🔍 JSON Lens

**JSON Lens** is a high-performance, production-ready JSON Formatter and Validator built with Next.js, TypeScript, and Tailwind CSS. It features a premium UI with glassmorphism, real-time validation, and a sleek dark mode.

## ✨ Features

- **🚀 Instant Formatting**: Pretty-print messy JSON strings instantly.
- **🛡️ Schema Validation**: Validate JSON against custom schemas using AJV.
- **🎨 Premium UI**: Modern aesthetic with Lucide icons, Framer Motion animations, and ShadCN components.
- **🌓 Theme Support**: Seamless switching between light and dark modes.
- **📝 Syntax Highlighting**: Beautiful code blocks powered by prism-react-renderer.
- **🛠️ Developer-First**: Built with TypeScript for type safety and maintainability.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Validation**: [AJV](https://ajv.js.org/)
- **Editor**: [Prism React Renderer](https://github.com/FormidableLabs/prism-react-renderer)

## 🏁 Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `/app`: Next.js App Router pages and layouts.
- `/components`: Reusable UI components (ShadCN + custom).
- `/lib`: Utility functions and shared logic.
- `/public`: Static assets.
